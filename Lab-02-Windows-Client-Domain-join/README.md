# Lab 02 – Windows Client Domain Join

## Objective
Create a Windows client virtual machine, configure its network settings, and join it to the Active Directory domain created in Lab 01.

## Environment
- Windows Server (Domain Controller)
- Windows Client (Virtual Machine)
- Active Directory Domain Services (AD DS)
- DNS (AD-integrated)
- VirtualBox (virtualized lab)

## Key Tasks Performed
- Creation of a Windows client virtual machine
- Configuration of VM hardware resources
- Network configuration for Domain Controller and client
- Assignment of static IP and DNS configuration
- Verification of DNS name resolution
- Successful domain join
- Domain user logon validation

## Outcome
- Windows client successfully joined to the domain
- DNS communication between client and Domain Controller verified
- Domain authentication confirmed on client workstation

## Notes
This lab builds directly on **Lab 01 – Active Directory and DNS Setup** and is part of a progressive enterprise Active Directory lab series.
